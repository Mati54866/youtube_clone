# Build and Deploy a Modern YouTube Clone Application in React JS with Material UI 5

![YouTube](https://i.ibb.co/4R5RkmW/Thumbnail-5.png)

A modern, fully responsive YouTube clone application built with React JS and Material UI 5. This project replicates core YouTube functionalities with a sleek, professional interface.

## ✨ Overview

This YouTube Clone application allows users to browse videos, search for content, and view video details with a seamless user experience. Built following industry best practices with modern React patterns.

## 🛠️ Tech Stack

- **React JS** - Frontend library
- **Material UI 5** - Component library & styling
- **React Router** - Navigation
- **Axios** - API requests
- **RapidAPI** - YouTube Data API integration

## 🔋 Key Features

- ✅ Browse trending videos
- ✅ Search for videos by keyword
- ✅ View video details with recommendations
- ✅ Channel pages with video lists
- ✅ Responsive design for all devices
- ✅ Category-based video browsing
- ✅ Dark/Light theme support
- ✅ Debounced search functionality

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- RapidAPI Key (YouTube v3 API)

### Installation

```bash
# Clone the repository
git clone https://github.com/Mati54866/youtube-clone.git

# Navigate to project
cd youtube-clone

# Install dependencies
npm install

# Create .env file and add your API key
echo "REACT_APP_RAPID_API_KEY=your_api_key_here" > .env

# Start development server
npm start
